# ShootingGame_DxLib

## 概要
DxLibで制作した2Dシューティングゲームです。
敵を倒して、ゲームオーバーまでスコアを稼ぐゲームです。
敵と弾の種類(色)が一致している場合のみダメージを与えることができます。

## 使用技術
-C++
-DxLib
-Visual Studio 2022

## 操作説明
-矢印キー : 移動
-スペースキー : ショット
-弾の切り替え : キーボード上部の1～4キー

## 工夫した点
-static const constexpr などを用い共通データの管理や、値変更不可によるバグ防止など安全性を高める設計。
-ユーティリティクラスを使用し、当たり判定計算(共通処理)を再利用できる設計にした。
-EnemyManagerクラスで構造体を用い、ステータスが違う敵をランダムに生成できるようにした。
-自機が発射する弾と敵の種類(色)が一致時のみダメージを与えられるようにした。

## 実行方法
zipを解凍後、Game.exe を実行してください。
